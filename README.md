# fdf
Fdf is a program that creates wireframe representation of various connected points via segments. <b>(Only macOS is supported).</b>

# Compiling
```minilibx``` is required

Run ```make```

# Usage
Run ```./fdf [file]```

# Example
```
./fdf maps/mars.fdf
```
